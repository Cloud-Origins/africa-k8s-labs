# Helm Packaging

Track: KCNA, CKAD  |  Estimated time: 2.5h  |  Primary tool: Helm  |  Type: Lab

Grading: manual. There is no autograder for this lab; your mentor reviews your show-off checkpoint in your PLG space.

## Objective

Package, install, upgrade and roll back an application with Helm.

## Your task

Run 'helm create' to scaffold a chart, parameterize the image and replica count in values.yaml, install the release, change a value and 'helm upgrade', then 'helm rollback' to the previous revision.

## Show-off checkpoint

Share 'helm history <release>' showing at least one upgrade and one rollback.
