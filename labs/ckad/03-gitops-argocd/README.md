# GitOps with ArgoCD

Track: CKAD  |  Estimated time: 3h  |  Primary tool: ArgoCD  |  Type: Lab

Grading: manual. There is no autograder for this lab; your mentor reviews your show-off checkpoint in your PLG space.

## Objective

Deploy declaratively from Git using ArgoCD (pull-based GitOps).

## Your task

Install ArgoCD, create an Application that points at a Git repo path containing Kubernetes manifests, sync it, then commit a change to Git and watch ArgoCD reconcile the cluster to match.

## Show-off checkpoint

Share the ArgoCD UI showing your Application Synced and Healthy after a Git-driven change.
