# CI/CD with GitLab

Track: CKAD  |  Estimated time: 2.5h  |  Primary tool: GitLab  |  Type: Lab

Grading: manual. There is no autograder for this lab; your mentor reviews your show-off checkpoint in your PLG space.

## Objective

Automate build-and-deploy with a GitLab CI pipeline.

## Your task

Write a .gitlab-ci.yml with build and deploy stages that build an image and deploy it to your cluster; run it on a push.

## Show-off checkpoint

Share a green GitLab pipeline and the resulting running workload.
