# CI/CD with Jenkins

Track: CKAD  |  Estimated time: 3h  |  Primary tool: Jenkins  |  Type: Lab

Grading: manual. There is no autograder for this lab; your mentor reviews your show-off checkpoint in your PLG space.

## Objective

Build and deploy to Kubernetes from a Jenkins pipeline.

## Your task

Run Jenkins, create a pipeline that builds a container image and applies Kubernetes manifests (kubectl/Helm) to your cluster on each commit.

## Show-off checkpoint

Share a successful Jenkins pipeline run log ending in a rolled-out Deployment.
