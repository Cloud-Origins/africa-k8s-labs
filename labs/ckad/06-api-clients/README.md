# Kubernetes API Clients

Track: CKAD  |  Estimated time: 2.5h  |  Primary tool: client libraries  |  Type: Lab

Grading: manual. There is no autograder for this lab; your mentor reviews your show-off checkpoint in your PLG space.

## Objective

Talk to the Kubernetes API from code using an official client library.

## Your task

Pick Go (client-go), Python (kubernetes) or Java (fabric8). Write a small program that authenticates via kubeconfig and lists all pods in a namespace with their phase.

## Show-off checkpoint

Share your program's source and its output listing pods and statuses.
