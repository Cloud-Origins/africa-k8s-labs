# Node Autoscaling (Karpenter)

Track: Elective  |  Estimated time: 2.5h  |  Primary tool: Karpenter  |  Type: Lab

Grading: manual. There is no autograder for this lab; your mentor reviews your show-off checkpoint in your PLG space.

## Objective

Autoscale nodes just-in-time (cloud provider required; cannot run on local kind).

## Your task

On a cloud cluster (EKS), install Karpenter, define a NodePool, and schedule unschedulable pods to trigger fast node provisioning.

## Show-off checkpoint

Share Karpenter provisioning a node within seconds of a pending pod.
