# Developer Portal (Backstage)

Track: Elective (optional)  |  Estimated time: 3h  |  Primary tool: Backstage  |  Category: Platform / Developer experience  |  Type: Lab

Grading: manual. This elective installs and drives real ecosystem tooling; your mentor reviews your show-off checkpoint. Some tools are heavy or need a cloud account, so they are optional and run at your own pace.

## Objective

Stand up an internal developer portal with a software catalog.

## Your task

Scaffold a Backstage app, register a component via catalog-info.yaml, and browse it in the catalog; optionally deploy Backstage to the cluster.

## Starting point (premade)

Install the tool, then build your task on top of it:

```bash
npx @backstage/create-app@latest
```

Follow the official quickstart for current versions and full steps: https://backstage.io/docs/getting-started/

## Show-off checkpoint

Share your Backstage catalog showing a registered component.
