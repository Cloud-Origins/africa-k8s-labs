# Operators

Track: Elective  |  Estimated time: 3h  |  Primary tool: Operator SDK  |  Type: Lab

Grading: manual. There is no autograder for this lab; your mentor reviews your show-off checkpoint in your PLG space.

## Objective

Extend Kubernetes with a custom controller and CRD.

## Your task

Using the Operator SDK (or kubebuilder), scaffold an operator with a simple CRD and a reconcile loop that creates a Deployment for each custom resource; deploy and test it.

## Show-off checkpoint

Share a custom resource you created and the Deployment your operator reconciled from it.
