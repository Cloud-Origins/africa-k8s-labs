# Cluster Setup (kubeadm/k3s/k3d)

Track: Elective  |  Estimated time: 3h  |  Primary tool: kubeadm/k3s  |  Type: Lab

Grading: manual. There is no autograder for this lab; your mentor reviews your show-off checkpoint in your PLG space.

## Objective

Build a cluster from scratch rather than using a managed one.

## Your task

Stand up a cluster with kubeadm (or k3s/k3d), join at least one worker, install a CNI, and confirm nodes are Ready and pods schedule.

## Show-off checkpoint

Share 'kubectl get nodes -o wide' with all nodes Ready and a test pod Running.
