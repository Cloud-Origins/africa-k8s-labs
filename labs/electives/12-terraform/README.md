# Infrastructure as Code (Terraform)

Track: Elective  |  Estimated time: 2.5h  |  Primary tool: Terraform  |  Type: Lab

Grading: manual. There is no autograder for this lab; your mentor reviews your show-off checkpoint in your PLG space.

## Objective

Provision Kubernetes resources declaratively with Terraform.

## Your task

Use the Terraform Kubernetes/Helm providers to create a namespace and deploy an app; run plan/apply, then change a value and re-apply.

## Show-off checkpoint

Share 'terraform plan' output for a change and the applied result.
