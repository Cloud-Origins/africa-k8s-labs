# Serverless (Knative Serving)

Track: Elective (optional)  |  Estimated time: 3h  |  Primary tool: Knative  |  Category: Serverless  |  Type: Lab

Grading: manual. This elective installs and drives real ecosystem tooling; your mentor reviews your show-off checkpoint. Some tools are heavy or need a cloud account, so they are optional and run at your own pace.

## Objective

Run scale-to-zero HTTP services with Knative Serving.

## Your task

Install Knative Serving with a networking layer, deploy a Knative Service, and observe it scale to zero when idle and back up on request.

## Starting point (premade)

Install the tool, then build your task on top of it:

```bash
kn quickstart kind
```

Follow the official quickstart for current versions and full steps: https://knative.dev/docs/install/quickstart-install/

## Show-off checkpoint

Share your Knative Service scaling from zero to N on a burst of requests.
