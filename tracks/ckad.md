# CKAD Track

Certified Kubernetes Application Developer. This path assumes the KCNA foundation and focuses on designing, building, and deploying applications. Estimated total: ~40 hours (excluding the KCNA prerequisites you should already have done).

Before you start: complete [../SETUP.md](../SETUP.md), and ideally the [KCNA track](kcna.md) core labs first.
